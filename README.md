# Django Chatbot

A basic chatbot built using the Django framework.
The chatbot takes user input from a web interface and returns responses using simple keyword-based NLP logic.

## Technologies

* Python
* Django
* HTML, CSS
* JavaScript

## Run the Project

1. Install Django
2. Run `python manage.py runserver`
3. Open `http://127.0.0.1:8000/` in the browser.
